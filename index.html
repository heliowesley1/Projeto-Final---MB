Programação Web

Tenho conhecimentos básicos em programação web, com experiência introdutória tanto no front-end quanto no back-end. No front-end, sei como estruturar páginas utilizando HTML, aplicar estilos com CSS e adicionar interatividade com JavaScript. Já no back-end, estou aprendendo a desenvolver aplicações utilizando Node.js e Express js, criando rotas e manipulando dados com estruturas simples.

Ferramentas e Tecnologias

Para versionamento de código e colaboração em projetos, utilizo Git e GitHub, garantindo organização e histórico de desenvolvimento. Meu ambiente de desenvolvimento principal é o Visual Studio Code, que me permite programar com agilidade por meio de extensões e recursos inteligentes. Além disso, tenho familiaridade com ferramentas como o Postman para testar requisições HTTP e o Figma para análise de protótipos e organização visual de interfaces.
